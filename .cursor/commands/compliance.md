# Compliance Review

Review crypto regulatory compliance. Apply to the user's request.

## Before Starting

Read `.claude/docs/legal/COMPLIANCE.md` for current compliance status.

## Process

1. Identify which regulations apply (MiCA, AML/KYC, GDPR, etc.)
2. Assess current compliance status against requirements
3. Flag areas requiring professional legal review
4. Create checklist of action items with priority

## Acceptance Criteria

- [ ] Applicable regulations identified
- [ ] No legal claims without qualification
- [ ] Professional review recommended where needed
- [ ] Risk disclaimers appropriate for DeFi context

## Output

Return: compliance assessment with checklist and action items.
