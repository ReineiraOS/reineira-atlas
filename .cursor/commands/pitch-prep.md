# Pitch Preparation

Prepare materials for investor conversations and fund introductions. Apply to the user's request.

## Before Starting

Read these files:
- `.claude/docs/strategy/BUSINESS_MODEL.md` — revenue model
- `.claude/docs/strategy/ROADMAP.md` — milestones
- `.claude/docs/intelligence/METRICS.md` — traction

## Process

1. Assess readiness against criteria (product, business model, metrics, team, materials)
2. Identify gaps and create action items
3. Prepare requested materials (deck outline, one-pager, talking points)
4. Position FHE advantage for the target audience

## Acceptance Criteria

- [ ] Readiness criteria assessed honestly
- [ ] Materials clear for non-technical audience
- [ ] Numbers are real, not projected (or clearly labeled)
- [ ] FHE positioning is crisp and differentiated

## Output

Return: pitch materials or readiness assessment with action items.
