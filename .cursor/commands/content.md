# Content Creation

Create content for the venture — tutorials, blogs, social threads. Apply to the user's request.

## Before Starting

Read `.claude/docs/growth/COMMUNITY_STRATEGY.md` for target audience and channels.

## Process

1. Determine content type (tutorial, blog, thread, documentation)
2. Research topic and gather accurate protocol details
3. Write draft following the appropriate structure:
   - Tutorial: Why → Setup → Code → Deploy → Verify
   - Blog: Hook → Problem → Solution → CTA
   - Thread: Hook → Key points → CTA
4. Ensure code samples compile and work
5. Apply brand voice: technical but accessible, builder-focused, show don't tell

## Acceptance Criteria

- [ ] Technically accurate (protocol details, code samples)
- [ ] Code compiles and works as written
- [ ] SEO basics (keyword in title, H2s)
- [ ] Brand voice is builder-focused and technical
- [ ] CTA is clear and relevant

## Output

Return: publishable content draft.
